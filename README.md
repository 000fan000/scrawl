scrawl
======
web capture crawler

Requirements
------------

- Ubuntu or other Debian-like OS
- latest version node.js


Install PhantomJS & CasperJS & bower
------------------------------------

    npm install -g phantomjs
    npm install -g casperjs
    npm install -g bower


Install
-------

    git clone https://github.com/antirek/scrawl.git
    cd scrawl
    npm install
    bower install
    npm start
